# BAZ Service Card Templates

3 reusable SVG templates. Each one is for a different pillar and visual tone.

## Files

- `template-strategy.svg` — Cream background, dark text. Use for: Strategy, Brand, Content, Lifecycle, Email, CRM, Analytics, AI.
- `template-performance.svg` — Dark background, light text. Use for: Performance Marketing, CRO, Paid Media.
- `template-seo.svg` — Red background, light text. Use for: SEO, Organic, PR, Influencer, Video, Affiliate, ABM, Social, Internationalization, Market Research.

That covers all 18 services. Pick the visual tone that fits the channel.

## Customize per service

Open the SVG in any editor. Change these 5 elements:

1. **Eyebrow** (top): change `PILLAR 0X · OWNED` → `SERVICE 04 · BRAND` (or whatever)
2. **Headline**: change the two-line title to your service name
3. **Subhead**: change the one-sentence description
4. **3 proof points**: replace the three number+label pairs with your proof (KPIs, case study numbers, time-to-result)
5. **Footer label**: change `BAZ · STRATEGY` → `BAZ · BRAND IDENTITY`

The visual system (color, type, layout, accent strip, BAZ mark) stays the same.

## Use in a deck or carousel

- Each card is 600×800 (3:4 ratio). Slides into a 3-card or 6-card carousel naturally.
- For LinkedIn: export each as PNG and post 1-3 at a time.
- For email: link to a hosted version (Vercel, GitHub Pages).
- For sales decks: drop into the pitch deck at the relevant slide.

## Don't

- Don't change the BAZ mark on the card. Always the red square + serif B.
- Don't use more than 3 proof points. Density kills.
- Don't mix this card with the marketing site visuals. Same system, same restraint.
